# 人口老龄化模拟器
作者：Alexander-code-bit-design
这是一个用于观察人口老龄化过程的交互式网页模拟器。你可以调整出生率、平均寿命和退休年龄，查看总人口、自然增长率和老龄人口比例如何变化。

## 文件说明

- `simulation.zip`：离线 local 版本压缩包，里面是 `population-aging-local.html`, 可直接在浏览器中运行。

## 使用方法

1. 下载仓库所有文件。
2. 解压 `simulation.zip`，然后打开 `population-aging-local.html`。

## 模拟参数

- 出生率：控制每秒新增人口数量。
- 平均寿命：控制人口在时间轴上的最大年龄。
- 退休年龄：控制人口进入老龄阶段的年龄线。

## 指标

- 自然增长率：根据最近几秒的人口变化估算。
- 老龄人口比例：退休年龄及以上人口占总人口的比例。
- 总人口：当前画布中仍在模拟的人口数量。

## 备注
